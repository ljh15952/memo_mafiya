# 《등 뒤의 메모》 기능 제안서

> GitHub 이슈 / README / 개발 회의 자료용 기능 제안 문서  
> 본 문서는 게임의 핵심 구조인 **메모 3장 사망**, **마지막 생존자 승리**, **마피아의 조작 능력**을 강화하기 위한 기능 제안 모음이다.

---

## 0. 전제 규칙

현재 게임 방향은 다음을 기준으로 한다.

- 플레이어는 등에 메모가 **3장 붙으면 사망**한다.
- 모든 플레이어의 최종 목표는 **마지막까지 살아남는 것**이다.
- 마지막까지 살아남는다는 것은 결과적으로 **자신을 제외한 모든 플레이어를 메모로 죽이는 것**을 의미한다.
- 시민과 마피아 모두 승리 조건은 동일하게 **최후의 생존자**가 되는 것이다.
- 마피아는 시민보다 강한 조작 능력을 가지며, 게임 분위기를 흔드는 역할을 한다.

---

## 1. 마피아 유전 기능

### 1.1 기능 개요

**마피아 유전**은 현재 마피아가 사망했을 때, 일정 시간 후 새로운 마피아가 생기는 기능이다.

이 기능의 목적은 마피아가 너무 빨리 죽었을 때 게임의 긴장감이 사라지는 문제를 막는 것이다.

기존 구조에서는 마피아가 초반에 죽으면 이후 게임이 단순한 시민 개인전으로 변할 수 있다.  
마피아 유전 기능을 넣으면, 플레이어들은 마피아가 죽은 뒤에도 계속 긴장해야 한다.

---

### 1.2 기본 규칙

| 항목 | 내용 |
|---|---|
| 발동 조건 | 현재 마피아 사망 |
| 알림 방식 | 모든 플레이어에게 마피아 사망 알림 출력 |
| 대기 시간 | 10~20초 |
| 신규 마피아 선정 | 생존자 중 1명 랜덤 선정 |
| 기존 마피아 | 사망 상태 유지 |
| 신규 마피아 공개 여부 | 비공개 |
| 반복 가능 여부 | 룸 설정으로 조절 |

---

### 1.3 진행 예시

1. 초록 마피아가 메모 3장으로 사망한다.
2. 모든 플레이어에게 알림이 표시된다.

```text
마피아가 사망했습니다.
15초 후 새로운 마피아가 선택됩니다.
```

3. 15초 동안 모든 플레이어는 서로를 의심한다.
4. 생존자 중 1명이 새로운 마피아가 된다.
5. 신규 마피아에게만 별도 알림이 표시된다.

```text
당신은 새로운 마피아가 되었습니다.
색 위조 능력이 활성화됩니다.
```

6. 다른 플레이어들은 누가 새 마피아가 되었는지 알 수 없다.

---

### 1.4 기대 효과

- 마피아가 초반에 죽어도 게임 긴장감이 유지된다.
- 플레이어들은 마피아 사망 후에도 안심할 수 없다.
- “방금 누가 마피아가 되었는가?”라는 새로운 추리가 생긴다.
- 기존 마피아를 죽인 플레이어도 다음 마피아가 될 수 있어 배신감이 생긴다.
- 후반부까지 색 위조와 분위기 조작 플레이가 유지된다.

---

### 1.5 밸런스 옵션

| 옵션 | 설명 |
|---|---|
| 유전 OFF | 마피아가 죽으면 더 이상 마피아 없음 |
| 1회만 유전 | 첫 번째 마피아 사망 시 한 번만 발동 |
| 무한 유전 | 마피아가 죽을 때마다 새 마피아 선정 |
| 후반 유전 금지 | 생존자 3명 이하에서는 유전 비활성화 |
| 위험자 우선 제외 | 등에 메모 2장 붙은 플레이어는 신규 마피아 후보에서 제외 |

### 추천 기본값

```text
마피아 유전: ON
유전 횟수: 1회
대기 시간: 15초
생존자 3명 이하: 유전 비활성화
```

## 2. 아이템 기능

### 2.1 기능 개요

아이템은 게임이 단순한 추격전으로 변하는 것을 막고, 플레이어가 상황을 뒤집을 수 있게 만드는 장치다.

현재 구조에서 플레이어가 계속 도망치기만 하면 게임이 술래잡기처럼 변할 위험이 있다.  
따라서 아이템은 다음 목적을 가진다.

- 도망만 치는 플레이어를 견제한다.
- 추격하는 플레이어에게 변수를 준다.
- 위험 상태인 플레이어에게 생존 수단을 준다.
- 마피아와 시민 모두에게 심리전 선택지를 제공한다.
- 마지막 생존 경쟁을 더 전략적으로 만든다.

---

## 3. 아이템: 카메라

### 3.1 기능 개요

카메라는 특정 순간을 촬영하고, 그 사진을 **사진 메모**로 만들어 다른 플레이어의 등에 붙일 수 있는 아이템이다.

사진은 완전한 증거가 아니라, 해석이 필요한 단서로 작동해야 한다.

---

### 3.2 기본 규칙

| 항목 | 내용 |
|---|---|
| 사용 방식 | 전방을 향해 사진 촬영 |
| 결과물 | 사진 메모 생성 |
| 부착 방식 | 일반 메모처럼 다른 플레이어의 등에 부착 |
| 사용 횟수 | 1~2회 |
| 사진 품질 | 약간 흐림 |
| 메모 색상 | 촬영자 색상 적용 |
| 마피아 사용 | 가능 |
| 마피아 색 위조 | 사진 메모에도 적용 가능 여부는 밸런스 옵션 |

---

### 3.3 사진 메모에 담기는 정보

사진 메모는 다음 정도의 정보만 제공한다.

- 누가 어느 공간에 있었는지
- 누가 누군가의 뒤에 있었는지
- 누가 도망가고 있었는지
- 누가 메모를 붙이려는 자세였는지
- 특정 플레이어들이 함께 있었는지

단, 다음 정보는 명확히 보여주지 않는 것을 추천한다.

- 실제 메모 작성자
- 실제 메모 부착 성공 여부
- 마피아 여부
- 색 위조 여부
- 정확한 촬영 시간

---

### 3.4 기대 효과

- 말뿐인 증언보다 강한 단서가 생긴다.
- 마피아가 진실 일부를 이용해 분위기를 조작할 수 있다.
- 시민도 마피아의 움직임을 제한적으로 기록할 수 있다.
- 사진이 붙은 플레이어는 강한 의심 대상이 된다.

---

## 4. 아이템: 로프

### 4.1 기능 개요

로프는 카우보이처럼 던져서 다른 플레이어의 움직임을 잠시 멈추게 하는 아이템이다.

이 기능은 도망만 치는 플레이어 때문에 게임이 단순한 술래잡기가 되는 문제를 막기 위한 핵심 아이템이다.

---

### 4.2 기본 규칙

| 항목 | 추천값 |
|---|---:|
| 사거리 | 중거리 |
| 투척 방식 | 조준 후 발사 |
| 명중 시 효과 | 이동 불가 또는 이동 속도 대폭 감소 |
| 지속 시간 | 2~3초 |
| 사용 횟수 | 1회 |
| 빗나감 | 아이템 소모 |
| 피격자 행동 | 메모 부착 불가, 이동 제한 |
| 사용자 행동 | 로프 유지 중 이동 속도 감소 |

---

### 4.3 사용 목적

- 도망가는 플레이어를 붙잡는다.
- 2장 위험 상태인 플레이어를 압박한다.
- 마피아로 의심되는 플레이어의 도주를 막는다.
- 마지막 메모를 붙일 기회를 만든다.
- 반대로 위험한 플레이어가 추격자를 묶고 도망갈 수도 있다.

---

### 4.4 밸런스 주의점

로프가 너무 강하면 한 번 잡힌 플레이어가 거의 확정적으로 죽는다.

따라서 다음 제한이 필요하다.

- 지속 시간은 짧아야 한다.
- 명중 난이도가 있어야 한다.
- 벽이나 장애물 뒤에서는 맞지 않아야 한다.
- 로프를 맞은 플레이어도 짧은 저항 입력으로 탈출 가능하게 할 수 있다.
- 같은 플레이어에게 연속 사용 제한을 둔다.

---

## 5. 아이템: 투명화 주스

### 5.1 기능 개요

투명화 주스는 일정 시간 동안 플레이어를 투명하게 만드는 아이템이다.

마피아 전용 능력으로만 쓰는 것이 아니라, 시민도 사용할 수 있는 아이템으로 만들면 개인전 생존 구조와 잘 맞는다.

---

### 5.2 기본 규칙

| 항목 | 추천값 |
|---|---:|
| 지속 시간 | 4~5초 |
| 사용 횟수 | 1회 |
| 이동 속도 | 90% |
| 완전 투명 여부 | 근거리에서는 실루엣 보임 |
| 메모 부착 | 가능, 단 부착 시작 시 투명 해제 추천 |
| 피격 가능 여부 | 로프 등 아이템에 피격 가능 |
| 밤 사용 | 불가 |

---

### 5.3 사용 목적

- 위험 상태에서 도망친다.
- 몰래 접근해 메모를 붙인다.
- 마피아처럼 보이도록 다른 플레이어를 혼란시킨다.
- 누가 접근했는지 알 수 없게 만들어 추리 난이도를 올린다.

---

### 5.4 밸런스 주의점

투명화는 강한 능력이므로 다음 제한을 추천한다.

- 완전 투명보다 반투명 실루엣이 좋다.
- 메모 부착 중에는 반드시 투명화가 해제되어야 한다.
- 투명화 상태에서는 아이템 사용을 제한할 수 있다.
- 연속 사용을 막기 위해 희귀 아이템으로 설정한다.

---

## 6. 아이템: 힐

### 6.1 기능 개요

힐은 등에 붙은 메모를 제거하는 아이템이다.

힐은 생존형 구조에서 가장 중요한 방어 아이템이다.  
등에 메모가 2장 붙은 플레이어가 즉시 사망하지 않고, 한 번 더 생존 전략을 세울 수 있게 한다.

---

### 6.2 기본 규칙

| 항목 | 내용 |
|---|---|
| 사용 대상 | 자신 또는 다른 플레이어 |
| 효과 | 등에 붙은 메모 1장 제거 |
| 사용 횟수 | 1회 |
| 제거 방식 | 가장 최근 메모 제거 추천 |
| 사용 시간 | 1.5~2초 |
| 사용 중 이동 | 불가 또는 느려짐 |
| 실패 조건 | 대상이 멀어지면 실패 |

---

### 6.3 자기 힐

자기 힐은 자신의 등에 붙은 메모를 1장 제거하는 방식이다.

다만 플레이어는 자기 등에 어떤 메모가 붙었는지 직접 볼 수 없기 때문에, 자기 힐을 쓰더라도 어떤 메모가 사라졌는지는 모른다.

이 구조는 기존 게임의 핵심인 “자기 등 정보 불확실성”을 유지한다.

---

### 6.4 타인 힐

타인 힐은 다른 플레이어의 등에 붙은 메모를 제거하는 방식이다.

이 기능은 협력처럼 보이지만, 개인전 구조에서는 배신과 거래의 도구가 된다.

예시:

```text
내 메모 하나 지워주면, 나도 너한테 안 붙일게.
```

또는:

```text
저 사람 살려두면 위험하니까 힐 쓰지 마.
```

---

### 6.5 밸런스 주의점

힐이 너무 많으면 메모 3장 사망 룰의 긴장감이 약해진다.

추천 제한은 다음과 같다.

- 맵에 적게 등장시킨다.
- 사용 시간이 있어야 한다.
- 힐 사용 모션이 주변에 보여야 한다.
- 힐은 1장만 제거한다.
- 제거되는 메모는 선택 불가 또는 가장 최근 메모로 고정한다.

## 7. 추가 아이템 제안

## 7.1 잉크 폭탄

### 기능

일정 범위 안의 플레이어 시야를 짧게 방해한다.

### 목적

- 추격에서 벗어난다.
- 메모를 붙이고 도망칠 시간을 만든다.
- 회의 직전 혼란을 만든다.

### 추천 사양

| 항목 | 추천값 |
|---|---:|
| 지속 시간 | 3초 |
| 범위 | 좁음 |
| 사용 횟수 | 1회 |
| 피아 구분 | 없음 |

---

## 7.2 가짜 메모지

### 기능

등에 붙어도 사망 카운트에는 포함되지 않는 가짜 메모를 생성한다.

### 목적

- 플레이어를 심리적으로 압박한다.
- 실제 메모 수를 헷갈리게 만든다.
- 마피아의 분위기 조작과 잘 어울린다.

### 밸런스

가짜 메모가 너무 강하면 실제 메모 추리가 무너질 수 있으므로, 가까이 보면 약간 다른 질감이 보이게 한다.

---

## 7.3 메모 방패

### 기능

일정 시간 동안 등에 메모가 붙는 것을 1회 막는다.

### 목적

- 2장 위험 상태에서 생존 기회를 준다.
- 마지막 생존 경쟁에서 방어 선택지를 만든다.
- 로프와 투명화에 대응할 수 있다.

### 추천 사양

| 항목 | 추천값 |
|---|---:|
| 지속 시간 | 8초 |
| 방어 횟수 | 1회 |
| 사용 모션 | 있음 |
| 희귀도 | 높음 |

---

## 7.4 메모 탐지기

### 기능

짧은 시간 동안 주변 플레이어의 메모 개수를 UI로 확인한다.

### 목적

- 자기 등이 아니라 주변 플레이어 상태를 빠르게 파악한다.
- 2장 위험 상태인 표적을 찾는다.
- 마피아가 사냥 대상을 고르는 데도 사용할 수 있다.

### 제한

- 메모 내용은 보이지 않는다.
- 실제 작성자는 보이지 않는다.
- 색 위조 여부도 보이지 않는다.

---

## 7.5 발소리 증폭기

### 기능

일정 시간 동안 주변 플레이어의 발소리를 크게 듣는다.

### 목적

- 누가 뒤에서 접근하는지 파악한다.
- 투명화 주스 사용자를 간접적으로 감지한다.
- 도망 플레이와 추격 플레이 모두에 사용 가능하다.

---

## 7.6 미끼 인형

### 기능

자신과 비슷한 실루엣의 더미를 짧은 시간 생성한다.

### 목적

- 추격자를 속인다.
- 로프를 낭비하게 만든다.
- 혼란 중에 도주하거나 접근한다.

### 밸런스

더미는 가까이 보면 가짜임을 알 수 있어야 한다.

---

## 7.7 접착제

### 기능

바닥에 설치하면 밟은 플레이어의 이동 속도가 짧게 감소한다.

### 목적

- 도주 경로를 막는다.
- 추격자를 늦춘다.
- 특정 장소 주변을 위험 지역으로 만든다.

---

## 7.8 메모 복사기

### 기능

다른 플레이어 등에 붙은 메모 중 1장을 복사해 새 메모로 만든다.

### 목적

- 같은 여론을 빠르게 퍼뜨린다.
- 특정 플레이어를 집단 표적으로 만든다.
- 진실인지 분탕인지 알 수 없는 정보가 확산된다.

### 주의점

복사된 메모는 원본과 구분되는 작은 표시가 있어야 한다.

---

## 7.9 해독 렌즈

### 기능

가까운 거리에서 메모 하나의 위조 가능성을 확인한다.

### 목적

- 마피아 색 위조에 대한 약한 대응 수단을 제공한다.
- 완전한 정답 공개가 아니라 추리 보조 역할만 한다.

### 추천 판정

```text
이 메모는 색이 수상해 보입니다.
```

정확히 “마피아가 썼다”라고 알려주면 안 된다.

---

## 7.10 위치 핑 메모

### 기능

메모에 특정 위치 정보가 자동으로 찍힌다.

예시:

```text
창고 근처에서 봄
```

### 목적

- 알리바이 추리에 도움을 준다.
- 마피아가 진실 일부를 이용해 분위기를 조작할 수 있다.
- 회의 때 논쟁거리가 생긴다.

## 8. 아이템 등급 제안

| Rare | 상황을 크게 바꾸는 아이템 | 로프, 카메라, 힐 |
| Epic | 강한 변수 아이템 | 투명화 주스, 메모 방패 |
| Legendary | 게임 후반 판도를 바꾸는 아이템 | 메모 복사기, 해독 렌즈 |

## 9. 아이템 등장 방식

### 9.1 맵 스폰 방식

아이템은 맵 곳곳에 랜덤으로 등장한다.

| 항목 | 추천값 |
|---|---:|
| 낮 시작 시 아이템 스폰 | 3~5개 |
| 희귀 아이템 수 | 0~1개 |
| 같은 위치 반복 | 제한 |
| 밤 스폰 | 없음 |

---

### 9.2 미니게임 보상 방식

일부 미니게임은 메모 대신 아이템을 보상으로 줄 수 있다.

예시:

```text
일반 미니게임 성공: 메모 1장
특수 미니게임 성공: 랜덤 아이템 1개
어려운 미니게임 성공: 힐 또는 카메라 확률 증가
```

## 10. 플레이어 레벨업 기능

### 10.1 기능 개요

플레이어는 게임을 반복하면서 경험치를 얻고 레벨업할 수 있다.

레벨업을 통해 기본 스펙이나 아이템 성능을 조금씩 강화할 수 있다.

이 기능의 목적은 장기 플레이 동기를 만드는 것이다.

---

### 10.2 성장 가능 항목

| 성장 항목 | 효과 |
|---|---|
| 이동 속도 | 기본 이동 속도 소폭 증가 |
| 메모 부착 속도 | 메모 붙이는 시간 소폭 감소 |
| 미니게임 속도 | 미니게임 제한 시간 보너스 |
| 아이템 쿨타임 | 일부 아이템 재사용 대기시간 감소 |
| 아이템 지속 시간 | 투명화, 방패 등 지속 시간 소폭 증가 |
| 아이템 보유 슬롯 | 보유 가능한 아이템 수 증가 |
| 힐 속도 | 메모 제거 시간 감소 |
| 로프 숙련도 | 로프 사거리 또는 투척 속도 소폭 증가 |

---

### 10.3 밸런스 원칙

레벨업은 장기 보상이어야 하지만, 신규 유저가 절대 이길 수 없는 구조가 되면 안 된다.

따라서 성장폭은 작아야 한다.

추천 원칙:

- 한 항목 최대 강화 수치는 5~10% 이내.
- 승패를 결정하는 절대 능력은 제공하지 않는다.
- 실력, 심리전, 위치 선정이 여전히 더 중요해야 한다.
- 랭크 모드에서는 레벨 보정을 끄는 옵션을 둘 수 있다.

---

### 10.4 성장 트리 예시

```text
생존 트리
- 이동 속도 +1%
- 힐 사용 속도 +3%
- 메모 방패 지속 시간 +0.5초

추격 트리
- 로프 투척 속도 +3%
- 메모 부착 속도 +2%
- 접착제 지속 시간 +0.3초

조작 트리
- 카메라 사용 후 메모 생성 속도 +5%
- 가짜 메모 지속 시간 +1초
- 투명화 주스 지속 시간 +0.3초
```

## 11. 전생 기능

### 11.1 기능 개요

전생은 플레이어가 일정 레벨에 도달했을 때 레벨업으로 얻은 모든 성장치를 초기화하는 대신, 다른 플레이어에게 자랑할 수 있는 외형적 보상을 얻는 기능이다.

전생은 게임 능력보다 명예와 과시를 위한 시스템이다.

---

### 11.2 기본 규칙

| 항목 | 내용 |
|---|---|
| 전생 조건 | 최대 레벨 도달 |
| 전생 후 | 레벨 1로 초기화 |
| 성장치 | 모두 초기화 |
| 보상 | 외형, 칭호, 이펙트, 프로필 장식 |
| 게임 능력 | 직접적인 능력 보상 없음 |
| 반복 가능 여부 | 가능 |

---

### 11.3 전생 보상 예시

| 보상 종류 | 예시 |
|---|---|
| 칭호 | 1회 전생자, 등 뒤의 지배자, 메모 수집가 |
| 이름표 장식 | 금색 테두리, 네온 테두리 |
| LED 효과 | 특별한 눈빛, 깜빡이는 왕관 아이콘 |
| 승리 포즈 | 마지막 생존자 전용 포즈 |
| 메모 스킨 | 고급 종이 질감, 검은 봉투, 황금 테이프 |
| 로비 배지 | 전생 횟수 표시 |
| 발자국 효과 | 짧은 잔상, 작은 별 효과 |

---

### 11.4 전생의 목적

- 오래 플레이한 유저에게 명예 보상을 준다.
- 능력 격차를 계속 누적시키지 않는다.
- 신규 유저와 고인물 간 밸런스 붕괴를 막는다.
- 플레이어가 자신의 플레이 시간을 과시할 수 있다.

## 12. 추가 시스템 제안

## 12.1 위험도 표시 시스템

다른 플레이어의 등에 붙은 메모 수에 따라 시각적 위험도를 표시한다.

| 메모 수 | 표시 |
|---:|---|
| 0장 | 표시 없음 |
| 1장 | 약한 주의 효과 |
| 2장 | 위험 테두리 또는 경고 사운드 |
| 3장 | 사망 |

자기 자신의 위험도는 직접 보이지 않아야 한다.  
다른 플레이어의 말이나 힐/탐지 아이템을 통해 간접적으로 알아야 한다.

---

## 12.2 현상금 시스템

특정 플레이어에게 메모가 2장 붙으면, 해당 플레이어를 죽인 사람에게 보너스를 준다.

예시 보상:

- 경험치 추가
- 아이템 드랍
- 다음 낮 시작 시 메모 1장 추가

주의점:

현상금이 너무 강하면 2장 플레이어가 무조건 사냥당하므로 보상은 작게 설정한다.

---

## 12.3 임시 동맹 시스템

플레이어 둘이 짧은 시간 동안 임시 동맹을 맺을 수 있다.

기능 예시:

- 서로의 메모 수를 UI로 확인 가능
- 서로에게 메모 부착 불가
- 60초 후 자동 해제
- 동맹 중 한 명이 배신 아이템을 사용하면 즉시 해제

개인전 구조에서 임시 동맹은 배신의 재미를 만든다.

---

## 12.4 배신 메모

동맹 중인 플레이어에게만 붙일 수 있는 특수 메모다.

배신 메모가 붙으면 일반 메모보다 강한 효과를 가진다.

예시:

```text
배신 메모는 2장으로 계산된다.
```

단, 너무 강하므로 커스텀 룸 옵션으로 분리하는 것이 좋다.

---

## 12.5 사망 리플레이

플레이어가 사망하면 마지막 10초를 짧게 보여준다.

단, 모든 정답을 공개하면 안 된다.

보여줄 수 있는 정보:

- 누가 근처에 있었는지
- 마지막 메모가 붙은 방향
- 사용된 아이템

숨겨야 하는 정보:

- 마피아 여부
- 색 위조 여부
- 실제 작성자 확정 정보

## 15. 최종 정리

이번 기능들의 핵심 목적은 다음과 같다.

1. **마피아 유전**으로 마피아 사망 후에도 긴장감을 유지한다.
2. **아이템**으로 게임이 단순한 술래잡기가 되는 것을 막는다.
3. **카메라**로 추리 단서를 강화한다.
4. **로프**로 도망 메타를 견제한다.
5. **투명화 주스**로 접근과 도주에 변수를 만든다.
6. **힐**로 2장 위험 상태에서 생존 선택지를 제공한다.
7. **레벨업**으로 장기 플레이 동기를 만든다.
8. **전생**으로 능력 격차를 초기화하면서도 자랑 요소를 제공한다.

최종적으로 이 기능들은 게임을 다음 방향으로 강화한다.

```text
단순히 도망치고 쫓는 게임이 아니라,
아이템과 메모, 진실과 거짓, 생존과 배신이 섞인
개인전 생존 심리 게임으로 만든다.
```
