## 1. 캐릭터 콘셉트

《등 뒤의 메모》의 캐릭터는 **큰 LED 머리와 단순한 몸체를 가진 마스코트형 캐릭터**다.

캐릭터의 가장 중요한 특징은 머리 전체가 LED 화면이라는 점이다. 플레이어는 이 LED 화면에 자유롭게 그림을 그리거나 표정을 표현할 수 있다.

캐릭터는 귀엽고 단순해야 하며, 멀리서 봐도 다음 정보가 바로 구분되어야 한다.

- 어떤 플레이어인지
- 어떤 색상의 플레이어인지
- 등에 메모가 붙어 있는지
- 현재 어떤 감정 표현을 하고 있는지
- 누가 접근해서 메모를 붙이려 하는지

캐릭터 디자인의 핵심 문장은 다음과 같다.

> 머리는 플레이어의 캔버스이고,  
> 등은 다른 플레이어가 남긴 평판이다.

---

## 2. 기본 실루엣

캐릭터는 머리가 크고 몸은 단순한 구조를 가진다.

### 기본 비율

| 부위 | 비율 | 설명 |
|---|---:|---|
| 머리 | 약 60~70% | 캐릭터의 핵심 시각 요소 |
| 몸체 | 약 20~25% | 단순한 옷 표현 중심 |
| 팔/손 | 약 5~10% | 메모 부착, 미니게임 조작 표현 |
| 다리/신발 | 약 5~10% | 이동 및 코스튬 표현 |

### 디자인 방향

- 머리가 매우 크다.
- 몸은 작고 단순하다.
- 전체적으로 장난감 같은 실루엣을 가진다.
- 캐릭터는 귀엽고 비폭력적인 인상을 준다.
- 마피아 게임이지만 공포보다는 파티게임 분위기를 우선한다.

## 3. 머리 사양

머리는 캐릭터의 가장 중요한 커스터마이즈 부위다.

플레이어는 머리의 형태를 변경할 수 있다. 머리 모양이 바뀌어도 LED 화면 기능은 유지된다.

### 기본 머리 형태

| 머리 형태 | 설명 |
|---|---|
| 둥근 사각형 | 기본 형태. LED 화면이 가장 잘 보이는 표준 머리 |
| 구형 | 공처럼 둥근 머리. 부드럽고 귀여운 인상 |
| 정육면체형 | 각진 네모 머리. 장난감 블록 같은 인상 |
| 캡슐형 | 위아래가 둥근 세로형 머리 |
| TV형 | 레트로 TV처럼 앞면이 넓은 머리 |
| 헬멧형 | 머리 전체를 감싸는 둥근 헬멧 느낌 |

### 머리 형태 변경 규칙

- 머리 형태는 외형 커스터마이즈 요소다.
- 게임 성능에 직접적인 차이를 주지 않는다.
- 모든 머리 형태는 LED 표시 영역을 가진다.
- 모든 머리 형태는 등에 메모가 붙은 상태를 방해하지 않아야 한다.
- 너무 작은 머리나 너무 얇은 머리는 사용하지 않는다.

### 추천 기본 머리

MVP에서는 다음 3종을 우선 적용한다.

| 우선순위 | 머리 형태 | 이유 |
|---:|---|---|
| 1 | 둥근 사각형 | LED 얼굴과 그림 표현이 가장 명확함 |
| 2 | 구형 | 귀엽고 캐주얼한 인상 |
| 3 | 정육면체형 | 실루엣 차이가 뚜렷함 |

## 4. LED 헤드 사양

캐릭터의 머리 전체는 LED 디스플레이로 구성된다.

플레이어는 LED 화면에 자유롭게 그림을 그리거나, 아이콘, 표정, 문양을 표시할 수 있다.

### LED 표시 영역

| 항목 | 설명 |
|---|---|
| 표시 위치 | 머리 전면 전체 |
| 표시 방식 | LED 도트 매트릭스 느낌 |
| 표시 색상 | 기본적으로 플레이어 색상과 연동 |
| 표현 요소 | 얼굴, 아이콘, 낙서, 감정 표현, 간단한 문자 |
| 표시 변경 | 플레이 중 실시간 변경 가능 |

### LED 표현 예시

- 웃는 얼굴
- 윙크 얼굴
- 화난 얼굴
- 슬픈 얼굴
- 왕관
- 별
- 하트
- 해골
- 고양이 얼굴
- 물음표
- 느낌표
- 화살표
- 간단한 손그림

### LED 그림 모드

플레이어는 LED 헤드에 직접 그림을 그릴 수 있다.

| 기능 | 설명 |
|---|---|
| 자유 드로잉 | 플레이어가 직접 선을 그린다 |
| 스탬프 | 준비된 아이콘을 배치한다 |
| 표정 프리셋 | 웃음, 화남, 의심, 슬픔 등 |
| 애니메이션 | 깜빡임, 반짝임, 흔들림 등 |
| 저장 기능 | 자주 쓰는 표정을 저장할 수 있음 |

### 제한 사항

- LED 그림은 게임 진행을 방해할 정도로 밝으면 안 된다.
- 다른 플레이어의 색상과 혼동되는 표시를 제한할 수 있다.
- 자유 그림은 악용 가능성이 있으므로 룸 설정에서 ON/OFF 가능하게 한다.
- MVP에서는 자유 드로잉보다 프리셋 표정과 간단한 아이콘을 우선 적용한다.

## 5. 플레이어 색상

각 플레이어는 고유 색상을 가진다.

플레이어 색상은 캐릭터의 몸, LED 표시, UI 아이콘, 메모 색상과 연결된다.

### 기본 색상

| 색상 | 설명 |
|---|---|
| 빨강 | 강한 인상, 의심 대상이 되기 쉬운 색 |
| 파랑 | 안정적이고 밝은 인상 |
| 노랑 | 눈에 잘 띄고 장난스러운 인상 |
| 초록 | 부드럽고 친근한 인상 |
| 보라 | 신비롭고 개성 있는 인상 |
| 주황 | 따뜻하고 활발한 인상 |
| 검정/회색 | 고급 설정 또는 특수 스킨용 |

### 색상 적용 부위

| 부위 | 색상 적용 여부 |
|---|---|
| LED 얼굴 | 적용 |
| 옷 기본 색상 | 적용 |
| 귀/측면 링 | 적용 |
| 신발 포인트 | 적용 |
| UI 이름표 | 적용 |
| 메모 색상 | 적용 |

### 중요 규칙

시민은 자기 색의 메모만 사용할 수 있다.  
마피아는 다른 플레이어의 색으로 위조 메모를 작성할 수 있다.

따라서 캐릭터 색상은 단순한 꾸미기 요소가 아니라, 게임의 추리 구조와 직접 연결되는 정보다.

## 6. 몸체 사양

몸체는 단순하고 귀여운 형태를 기본으로 한다.

머리가 시각적으로 가장 중요하므로, 몸체는 복잡하지 않아야 한다.

### 기본 몸체 구성

| 부위 | 설명 |
|---|---|
| 몸통 | 짧고 둥근 형태. 후드티 또는 단순한 옷 기반 |
| 팔 | 짧고 둥근 팔. 메모 붙이기 모션에 사용 |
| 손 | 장갑 형태. 색상 변경 가능 |
| 다리 | 짧고 단순한 다리 |
| 발 | 큰 신발 형태. 스킨 아이템 적용 가능 |

### 디자인 방향

- 몸체는 간단해야 한다.
- 머리보다 눈에 띄면 안 된다.
- 등에 메모가 잘 보여야 한다.
- 의상은 개성을 주되 실루엣을 크게 망치면 안 된다.
- 플레이어 색상이 항상 유지되어야 한다.

---

## 7. 등 메모 부착 영역

이 게임에서 등은 매우 중요한 정보 표시 영역이다.

플레이어는 자신의 등에 붙은 메모를 직접 볼 수 없고, 다른 사람의 등에 붙은 메모만 볼 수 있다.

따라서 캐릭터 디자인은 등에 메모가 잘 보이도록 설계되어야 한다.

### 등 영역 규칙

| 항목 | 규칙 |
|---|---|
| 메모 부착 위치 | 몸통 뒤쪽 또는 머리 뒤쪽 |
| 메모 가시성 | 다른 플레이어가 쉽게 볼 수 있어야 함 |
| 의상 간섭 | 망토, 큰 가방 등은 메모를 가리면 안 됨 |
| 모자 간섭 | 등 메모 영역을 가리지 않아야 함 |
| 카메라 거리 | 중거리에서도 메모 색이 구분되어야 함 |

### 메모 부착 가능 영역

MVP에서는 다음 영역을 추천한다.

1. 등 중앙
2. 머리 뒤쪽 하단
3. 후드티 등판 영역

### 메모 배치 방식

- 메모는 붙은 순서대로 정렬된다.
- 오래된 메모는 왼쪽 위에 배치된다.
- 최근 메모는 오른쪽 아래에 배치된다.
- 메모가 너무 많을 경우 일부가 겹쳐 보일 수 있다.

## 8. 외형 커스터마이즈

캐릭터는 다양한 외형 아이템으로 꾸밀 수 있다.

다만 커스터마이즈는 게임 판독성을 해치면 안 된다.

### 커스터마이즈 부위

| 부위 | 설명 |
|---|---|
| 머리 형태 | 구형, 사각형, 캡슐형 등 |
| LED 패턴 | 기본 도트, 별 패턴, 스캔라인 등 |
| 얼굴 표정 | 저장된 LED 표정 프리셋 |
| 옷 | 후드티, 점퍼, 조끼, 유니폼 등 |
| 신발 | 운동화, 부츠, 슬리퍼 등 |
| 장갑 | 기본 장갑, 두꺼운 장갑, 손가락 장갑 등 |
| 모자 | 캡, 비니, 왕관, 고깔모자 등 |
| 액세서리 | 배지, 이어링, 작은 장식 등 |

---

## 8.1 머리 커스터마이즈

머리 형태는 캐릭터의 인상을 크게 바꾸는 주요 아이템이다.

| 아이템 | 설명 |
|---|---|
| 기본 LED 헤드 | 둥근 사각형 기본 머리 |
| 볼 헤드 | 구형 머리 |
| 큐브 헤드 | 네모난 블록형 머리 |
| 캡슐 헤드 | 세로로 긴 둥근 머리 |
| 레트로 TV 헤드 | 오래된 TV 느낌의 머리 |
| 스페이스 헬멧 | 우주 헬멧 느낌의 머리 |

머리 형태가 달라져도 LED 기능과 플레이어 색상은 유지된다.

---

## 8.2 의상 커스터마이즈

몸체 의상은 플레이어의 개성을 표현하는 요소다.

| 의상 | 설명 |
|---|---|
| 기본 후드티 | 기본 의상 |
| 점퍼 | 캐주얼한 외형 |
| 작업복 | 미니게임 담당자 느낌 |
| 탐정 코트 | 추리 게임 분위기 강화 |
| 우주복 | SF 느낌 강화 |
| 파티복 | 이벤트용 의상 |

의상은 플레이어 색상을 기본으로 유지하되, 일부 포인트 색상만 변경할 수 있다.

---

## 8.3 신발 커스터마이즈

신발은 캐릭터 하단의 개성을 표현하는 아이템이다.

| 신발 | 설명 |
|---|---|
| 기본 운동화 | 기본 신발 |
| 하이탑 운동화 | 캐주얼 스타일 |
| 둥근 부츠 | 장난감 같은 느낌 |
| 슬리퍼 | 코믹한 분위기 |
| 빛나는 신발 | 이동 시 작은 빛 효과 |
| 스케이트 신발 | 이동 애니메이션 특화 가능 |

신발은 게임 판독성에 큰 영향을 주지 않으므로 비교적 자유롭게 꾸밀 수 있다.

---

## 8.4 장갑 커스터마이즈

장갑은 메모 붙이기, 미니게임 조작, 카메라 사용 등 손 동작에서 보인다.

특히 1인칭 시점에서 플레이어 손이 보일 경우 장갑 아이템은 중요하다.

| 장갑 | 설명 |
|---|---|
| 기본 장갑 | 표준 검정 장갑 |
| 컬러 장갑 | 플레이어 색상과 연결된 장갑 |
| 두꺼운 장갑 | 귀엽고 둥근 인상 |
| 손가락 장갑 | 세밀한 조작 느낌 |
| 고무 장갑 | 코믹한 느낌 |
| 반짝이 장갑 | 이벤트성 아이템 |

---

## 8.5 모자 커스터마이즈

모자는 머리 위에 장착되는 장식 아이템이다.

단, LED 화면을 가리면 안 된다.

| 모자 | 설명 |
|---|---|
| 캡 | 기본 캐주얼 모자 |
| 비니 | 부드러운 인상 |
| 왕관 | 장난스러운 강조 아이템 |
| 고깔모자 | 파티 분위기 |
| 탐정 모자 | 추리 테마 아이템 |
| 작은 안테나 | 로봇 느낌 강화 |

### 모자 제한 규칙

- LED 화면을 가리면 안 된다.
- 플레이어 이름표를 가리면 안 된다.
- 등 메모를 가리면 안 된다.
- 너무 큰 모자는 히트박스 오해를 만들 수 있으므로 제한한다.

## 9. 상점 및 랜덤 뽑기 시스템

캐릭터 꾸미기 아이템은 상점에서 획득할 수 있다.

기본 구조는 **랜덤 뽑기 방식**이다.

플레이어는 게임 플레이 보상 또는 재화를 사용해 랜덤 박스를 열고, 머리 형태, 신발, 장갑, 모자, 의상 등을 획득한다.

### 상점 구조

| 항목 | 설명 |
|---|---|
| 일반 상점 | 기본 아이템 구매 |
| 랜덤 박스 | 무작위 아이템 획득 |
| 시즌 상점 | 기간 한정 아이템 판매 |
| 이벤트 상점 | 이벤트 보상 교환 |
| 중복 보상 | 중복 아이템은 조각 또는 재화로 변환 |

### 랜덤 뽑기 대상

| 카테고리 | 포함 여부 |
|---|---|
| 머리 형태 | 포함 |
| 의상 | 포함 |
| 신발 | 포함 |
| 장갑 | 포함 |
| 모자 | 포함 |
| LED 표정 프리셋 | 포함 |
| LED 효과 | 포함 |
| 메모 스티커 테마 | 선택 포함 |

## 9.1 랜덤 뽑기 예시

플레이어가 랜덤 박스를 열면 다음 중 하나를 획득할 수 있다.

| 결과 | 예시 |
|---|---|

## 10. 아이템 등급

아이템은 희귀도에 따라 등급을 나눌 수 있다.

| 등급 | 설명 | 예시 |
|---|---|---|
| Common | 기본 아이템 | 기본 운동화, 기본 장갑 |
| Rare | 색상이나 형태가 조금 특별한 아이템 | 하이탑 운동화, 비니 |
| Epic | 눈에 띄는 개성 아이템 | 큐브 헤드, 탐정 코트 |
| Legendary | 강한 시각 효과 또는 희귀 외형 | 스페이스 헬멧, 왕관 |
| Event | 이벤트 기간 한정 아이템 | 할로윈 모자, 겨울 장갑 |

### 중요 규칙

아이템 등급은 외형 희귀도만 의미한다.  
게임 능력에는 영향을 주면 안 된다.

## 11. 게임플레이 판독성 규칙

커스터마이즈는 자유롭게 제공하되, 추리 게임의 핵심 정보를 방해하면 안 된다.

### 반드시 유지해야 하는 정보

| 정보 | 이유 |
|---|---|
| 플레이어 색상 | 메모 색상 추리에 필요 |
| 등 메모 | 게임의 핵심 정보 |
| LED 표정 | 감정 표현 및 커뮤니케이션 |
| 이름표 | 플레이어 식별 |
| 접근 모션 | 메모 부착 여부 확인 |

### 금지 또는 제한해야 하는 외형

| 외형 | 제한 이유 |
|---|---|
| 등을 완전히 가리는 망토 | 메모 확인 불가 |
| 너무 큰 날개 | 시야 방해 |
| LED 화면을 가리는 모자 | 캐릭터 핵심 기능 손상 |
| 다른 플레이어 색상으로 오해되는 의상 | 색상 추리 방해 |
| 지나치게 밝은 이펙트 | 밤 투표 시 가독성 저하 |
| 너무 큰 액세서리 | 히트박스와 시야 오해 발생 |

## 12. MVP 적용 범위

초기 개발 버전에서는 캐릭터 커스터마이즈를 너무 많이 넣지 않는다.

MVP에서는 게임의 핵심인 메모, 색상, LED 얼굴이 잘 보이는 것이 가장 중요하다.

### MVP 포함 항목

| 항목 | 포함 여부 |
|---|---|
| 기본 둥근 사각형 머리 | 포함 |
| 구형 머리 | 포함 가능 |
| 정육면체형 머리 | 포함 가능 |
| 플레이어 색상 4~6종 | 포함 |
| 기본 후드티 | 포함 |
| 기본 신발 | 포함 |
| 기본 장갑 | 포함 |
| LED 표정 프리셋 | 포함 |
| 간단한 LED 드로잉 | 포함 |
| 등에 메모 표시 | 포함 |

### MVP 제외 항목

| 항목 | 제외 이유 |
|---|---|
| 복잡한 모자 | LED 화면을 가릴 가능성 있음 |

## 14. 최종 정리

《등 뒤의 메모》의 캐릭터는 단순한 아바타가 아니다.

캐릭터의 머리는 플레이어가 자유롭게 자신을 표현하는 LED 캔버스이며, 등은 다른 플레이어가 남긴 평판이 쌓이는 추리 공간이다.

따라서 캐릭터 사양에서 가장 중요한 것은 다음 세 가지다.

1. 머리가 크고 LED 표현이 잘 보여야 한다.
2. 몸은 단순하지만 꾸미기 요소가 있어야 한다.
3. 등 메모와 플레이어 색상은 어떤 스킨을 착용해도 명확히 보여야 한다.

캐릭터 커스터마이즈는 재미와 수집 요소를 제공하지만, 추리 게임의 판독성을 해치면 안 된다.

최종적으로 이 캐릭터 시스템은 다음 문장으로 정의할 수 있다.

> 플레이어는 머리에 자신을 그리고,  
> 다른 사람은 등에 평판을 붙인다.
